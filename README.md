# TIASim
TIASim - Transimpedance Amplifier Simulation

## op-amps

Open-loop gain, input-referred voltage and current noise, and input-capacitance are modeled for the following op-amps:

| Op-amp        | Input           | Bandwidth |
| ------------- | -------------   | --------- |
| OPA657        | FET             |  1.6 GHz  |
| OPA859        | FET             |  1.8 GHz  |
| OPA847        | BJT             |  3.9 GHz  |
| OPA858        | FET             |  5.5 GHz  |
| OPA855        | BJT             |  8 GHz    |
